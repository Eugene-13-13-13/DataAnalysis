# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "Смартфоны" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Анализ характеристик смартфонов для разделения устройств по различным типам, прогнозирования ценовых категорий и выявления рыночных закономерностей. Дано 15000 записей с различными характеристиками устройств, требующими очистки и стандартизации.  
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет или скрипт загрузки
